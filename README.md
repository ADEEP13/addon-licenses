## How to Use These Licenses

To apply a license to your project:

1. Choose the appropriate license
2. Copy the LICENSE file from this repository
3. Paste it into your project repository as `LICENSE`
4. Keep the copyright notice intact

Example project structure:

```
addon-project/
│
├── LICENSE
├── README.md
├── src/
```
